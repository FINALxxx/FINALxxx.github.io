# 飞行能量管理技术

## 起飞阶段

（注：以下假设人工调速，高度单位为ft）

- 地面预位水平导航/航向选择



- 起飞速度V2+10~V2+20(起飞推力)

- 400ft AGL预位垂直导航/高度，启用AP



- 固定减推力程序NADP1
  - 800ft AGL程序开始
  - 减推力爬升速度V2+10~V2+20(爬升推力)
  - 保持形态不变
  - 3000ft AGL程序结束
  - 按计划增速收襟翼
    - 先增速到光洁速度(UP速度)，同时收襟翼
    - 襟翼收上后增速到航路爬升速度(由飞行计划给出)
- 固定减推力程序NADP2
  - 800ft AGL程序开始
  - 减推力爬升速度V2+10~V2+20(爬升推力)
  - 按计划增速收襟翼，增速到光洁速度为止
  - 3000ft AGL程序结束
  - 按计划增速到航路爬升速度
- 固定减推力程序PROC A
  - 类似于NADP1，只是程序开始高度为1500ft AGL
- 固定减推力程序PROC B(国内常用，一千减推力收形态，三千增速爬升)
  - 类似于NADP2，只是程序开始高度为1000ft AGL



## 下降-进场-进近阶段

- 核心：能量管理，需要把能量减少到FAF点时，高度对应势能+襟翼5速度对应动能
- 计算需要减少的能量

  - FAF点剩余能量：
    - FAF高\*100+襟翼5速度\*10
    - 简化计算：机场高\*100+襟翼5速度\*10
    - 简化原因：因为FAF点到机场之间速度不变，而高度与能量减少量约为1：1
  - 需要减少的能量：
    - (当前高-机场高)\*100+(当前速度-襟翼5速度)\*10



- 当前能量 == 当前需要减少的能量时

  - 说明处于下降剖面上，可以按3°下滑角下降
  - 也就是让下降率==地速一半+50 

  

- 当前能量 =/= 当前需要减少的能量时

  - 1）需要速度固定时，减油门/拉减速板/放形态 = 压机头 = 下高度
  - 2）需要高度固定时，拉减速板/放形态 = 抬机头 = 减速度

  - 一般情况：限速时，可以下高；限高时，可以减速；都不限，可以先下高，再减速放形态（提前减速可能会导致机场拥堵）。
  - 特殊情况：如果由于高度太高使得总能量太高，可以先减速放形态后，再大下降率下高。但如果实在太高下不了，就申请盘旋下高或者申请做一个机动。



- 最终目标：一般在跑道第4边~FAF前2海里之间，高度下降到FAF点限制高度，调速到襟翼5速度，放形态到襟翼5
  - 繁忙机场可以适当延后减速点



## 最终进近-着陆阶段

- 非繁忙机场一般在FAF前2海里开始建立着陆构型，繁忙机场可以适当延后建立着陆构型点
- 核心：CDFA技术，保持3°下降剖面。减速至Vref+风修正，着陆形态放好
- 一旦目视机场，就要注意机场papi灯和接地区，而不是下降率。
- 400ft AGL切手动，不需要看下降率，只需要看接地姿态、速度、高度
- 20ft AGL一边慢慢拉平一边收油门（太快拉平可能会拉飘），当收光油门后正好拉平，稳住舵等下沉
