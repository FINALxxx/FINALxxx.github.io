## 3. Willing to share/contribute?
!!! info "前置条件"
    - 你需要能够访问Github，并且掌握一些Git知识，比如提交PR
    - 后续我们将设置镜像仓库到gitee，方便国内使用

- 使用Fork获取仓库并提交贡献（推荐）
    1. 找到[本仓库](https://github.com/FINALxxx)
    2. Fork仓库到自己的远程仓库（一次性）
    3. Clone远程仓库到本地（一次性）
    4. 每次修改前先Pull一次，然后在本地进行修改
    5. 修改完毕后，Push到自己的远程仓库
    6. 在Github上提交PullRequest
- 使用Clone获取仓库并提交贡献（不推荐）
