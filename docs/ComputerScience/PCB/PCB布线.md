# PCB布线

## 布线要求

- 通用走线规则
  - 同一层中钝角走线：锐角/直角走线会导致折脚的阻抗不连续，推荐走线135°
  - 布线尽量远离安装孔和板边缘
  - 过孔与导线的连接处需要添加泪滴
- 电源线规则
  - 电源线尽可能粗：25℃铜厚为10z的导线的前提下，10mil宽度可承载0.65A电流，40mil可承载2.3A电流
- 高频信号线规则
  - 高频线尽量短，做好与其他信号的屏蔽隔离（即铺铜）
  - 相邻层最好正交走线，避免平行走线，防止高频线对低频线的干扰
  - 由于时钟信号要求信号线等长（保证每条线的时钟是一致的），所以最好等长布线
  - 由于差分信号要求信号线等长等距紧耦合，所以最好差分对布线
  - 遵守3W原则：若线宽为W，则同层的两条线的间距必须大于等于3W





## 布线顺序

- 顶层优先
- 密度优先：优先从连线最密集的地方开始布线
- 关键元件优先：优先对核心模块布线
- 关键信号线优先 ：优先对高频/时钟/同步信号线、电源线布线
