# HOSC

**HNNU Open Source Community**

> 本项目是作者大学四年的学习笔记整理，同时开放 Community 板块供大家分享学习资料。

## 在线访问

🔗 [HOSC 网站](http://FINALxxx.github.io)

## 项目结构

```
docs/
├── Index/              # 目录索引
├── Curriculum/        # 课程笔记
│   ├── Android/        # Android 开发
│   ├── CN/             # 计算机网络
│   ├── CO/             # 计算机组成原理
│   ├── CT/             # 编译原理
│   ├── DB/             # 数据库
│   ├── DS/             # 数据结构
│   ├── Java/           # Java
│   ├── Math/           # 数学
│   ├── MP/             # 微机原理
│   ├── OS/             # 操作系统
│   ├── SE/             # 软件工程
│   └── Web/            # Web 开发
├── ComputerScience/    # 计算机科学专题
│   ├── FreeRTOS/       # 嵌入式操作系统
│   ├── LVGL/           # GUI 库
│   ├── PCB/            # PCB 设计
│   ├── STM32基础/      # STM32 基础
│   ├── STM32进阶/      # STM32 进阶
│   ├── 体系结构/       # 体系结构
│   ├── 原理图/         # 电路原理图
│   └── 数字电路/       # 数字电路
├── MyInterests/        # 课外兴趣
├── Community/          # 社区分享
├── About/              # 关于项目
└── Sandbox/            # 沙盒测试
```

## 技术栈

- **文档框架**: [MkDocs](https://www.mkdocs.org/)
- **主题**: Material with Shadcn
- **部署**: GitHub Pages

## 本地运行

```bash
pip install -r requirements.txt
mkdocs serve
```
