# HOSC

**HNNU Open Source Community**

> 本项目是作者大学四年的学习笔记整理，同时开放 Community 板块供大家分享学习资料。

## 在线访问

🔗 [HOSC 网站](http://FINALxxx.github.io)

## 项目结构

```
docs/
├── Index/              # 目录索引
├── Curriculum/        # 课程笔记
├── ComputerScience/    # 计算机科学专题
├── MyInterests/        # 课外兴趣
├── Community/          # 社区分享
├── About/              # 关于项目
└── Sandbox/            # 沙盒测试
```

## 技术栈

- **文档框架**: [MkDocs](https://www.mkdocs.org/)
- **主题**: Material with Shadcn
- **部署**: GitHub Pages

## 本地运行

```bash
pip install -r requirements.txt
mkdocs serve
```

## 参与贡献

1. Fork 本仓库
2. 克隆到本地
3. 在Community下添加或修改Markdown内容（可以通过mkdocs serve进行预览）
4. 提交 Pull Request

TIPS：我们在构建后将去除掉markdown源文件。
