# FOSN

> FINALx Open Source Notebook

这里是 FINALx 的个人学习笔记站点，记录大学四年所学知识。

## 本地预览

```bash
# 安装依赖
pip install -r requirements.txt

# 启动预览服务器
mkdocs serve
```

访问 `http://localhost:8000` 即可预览站点。
